About

Pheonix Browser is a local desktop app. GitHub Pages can’t run the Electron app in your browser, so this page is just a preview and instructions. To use the browser, clone or download the repo and run it locally.

Install & Run

  1. Install Node.js (LTS) and Git on your machine.
  2. Clone or download the repository.
  3. Open a terminal in the project folder and run:
npm install
npm start

On launch, you’ll see the animated Pheonix home with a Google-powered search.
